# BotBotlr
This is a game listing app
## To run in your local computer 
1. clone
2. open the directory 
    ```bash
    cd Bot-Battlr
    ```
3. Install depedancies
    ```bash
    npm install
    ```
4. Open the json mock server
    ```bash
    npm run server
    ```
5. Open the project on your browser
    ```bash
    npm run dev
    ```
